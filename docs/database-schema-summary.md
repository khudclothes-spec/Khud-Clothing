| Column | Type | Constraints |
|---|---|---|

| `price` | NUMERIC(10,2) | NOT NULL |

| `is_featured` | BOOLEAN | NOT NULL DEFAULT FALSE |
| `is_active` | BOOLEAN | NOT NULL DEFAULT TRUE |

**Notes:**

- References `categories` table (not defined in this SQL file — must exist separately).

### `orders`

| Column | Type | Constraints |
|---|---|---|
| `id` | UUID | PK, DEFAULT gen_random_uuid() |
| `profile_id` | UUID | NOT NULL, FK → profiles(id) ON DELETE CASCADE |
| `order_number` | TEXT | NOT NULL UNIQUE |
| `status` | TEXT | NOT NULL DEFAULT 'pending', CHECK IN ('pending','confirmed','processing','shipped','delivered','cancelled') |
| `subtotal` | NUMERIC(10,2) | NOT NULL |
| `shipping_cost` | NUMERIC(10,2) | NOT NULL DEFAULT 0 |
| `total_amount` | NUMERIC(10,2) | NOT NULL |
| `customer_name` | TEXT | NOT NULL |
| `customer_phone` | TEXT | NOT NULL |
| `customer_email` | TEXT | NOT NULL |
| `shipping_address` | TEXT | NOT NULL |
| `city` | TEXT | NOT NULL |
| `notes` | TEXT | nullable |
| `created_at` | TIMESTAMPTZ | NOT NULL DEFAULT NOW() |
| `updated_at` | TIMESTAMPTZ | NOT NULL DEFAULT NOW() |

**Notes:**
- User is identified by `profile_id` (not `user_id`).
- `order_number` must be provided; it is NOT auto-generated by DB.
- Shipping is a flat `shipping_address` TEXT + `city` — no separate line1/line2/state/postal columns.
- `customer_email` is required (NOT NULL).
- No `discount_amount`, `tax_amount`, `payment_method`, or `payment_status` columns.
- `total_amount` is the grand total field (not `total`).
- `updated_at` auto-updated via trigger `update_orders_updated_at`.

---

## Functions

### `update_updated_at_column()`
Sets `NEW.updated_at = NOW()` before any UPDATE. Used by products and orders triggers.

### `public.handle_new_user()`
SECURITY DEFINER trigger function. On every auth.users INSERT, creates a matching row in `public.profiles` with `id` and `full_name` (from `raw_user_meta_data->>'full_name'`).

## Row Level Security (RLS)

RLS is enabled on: `products`, `product_variants`, `product_media`, `orders`, `order_items`, `reviews`, `custom_orders`.

### Products

| Policy | Operation | Rule |
|---|---|---|
| "Public products read" | SELECT | `is_active = true` |
| _(no admin write policies)_ | INSERT/UPDATE/DELETE | Missing — must be added via migration |

### Product Variants & Product Media

| Policy | Operation | Rule |
|---|---|---|
| "Public variants read" | SELECT | `true` (all) |
| "Public product media read" | SELECT | `true` (all) |
| _(no admin write policies)_ | INSERT/UPDATE/DELETE | Missing |

### Orders

| Policy | Operation | Rule |
|---|---|---|
| "Users read own orders" | SELECT | `profile_id = auth.uid()` |
| "Users insert own orders" | INSERT | `profile_id = auth.uid()` |
| _(no admin or update policies)_ | UPDATE/DELETE | Missing |

### Order Items

| Policy | Operation | Rule |
|---|---|---|
| "Users read own order items" | SELECT | EXISTS order matching profile_id |
| _(no insert or admin policies)_ | INSERT/UPDATE/DELETE | Missing — users can't insert order items |

### Reviews

| Policy | Operation | Rule |
|---|---|---|
| "Public approved reviews" | SELECT | `approved = true` |
| "Authenticated users insert reviews" | INSERT (authenticated) | `profile_id = auth.uid()` |
| _(no admin or update policies)_ | UPDATE/DELETE | Missing |

### Custom Orders

| Policy | Operation | Rule |
|---|---|---|
| "Users insert custom orders" | INSERT (authenticated) | `profile_id = auth.uid()` |
| "Users read own custom orders" | SELECT | `profile_id = auth.uid()` |
| _(no admin policies)_ | UPDATE/DELETE | Missing |

## Storage Buckets

| Bucket ID | Name | Public |
|---|---|---|
| `product-images` | product-images | true |
| `custom-order-artwork` | custom-order-artwork | false |

### Storage Policies — product-images

| Policy | Operation | Rule |
|---|---|---|
| "Public can view product images" | SELECT | bucket_id = 'product-images' |
| "Admins can upload product images" | INSERT (authenticated) | bucket_id = 'product-images' AND profile.role = 'admin' |
| "Admins can update product images" | UPDATE (authenticated) | bucket_id = 'product-images' AND profile.role = 'admin' |
| "Admins can delete product images" | DELETE (authenticated) | bucket_id = 'product-images' AND profile.role = 'admin' |

### Storage Policies — custom-order-artwork

| Policy | Operation | Rule |
|---|---|---|
| "Users upload own artwork" | INSERT (authenticated) | bucket_id = 'custom-order-artwork' AND folder matches auth.uid() |
| "Users view own artwork" | SELECT (authenticated) | own folder OR admin role |
| "Users update own artwork" | UPDATE (authenticated) | own folder OR admin role |
| "Users delete own artwork" | DELETE (authenticated) | own folder OR admin role |
